# 动作

动作是构建整个插件体系的基本定义。

# 动作名的定义

我们将动作名归为三类`Cover`/`Concat`/`Calc`。

这三种分别表示如何修改这项内容。

* cover     只执行最后一个动作
* concat    执行所有动作，并且将所有action数据以数组返回
* calc      依次执行所有动作，并在执行中传入上一个动作的返回数据
