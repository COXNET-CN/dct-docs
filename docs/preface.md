# 前言

该文档将教会你如何使用 `data-collect-template`，简称`DCT`。

该框架是以插件、主题驱动的信息采集的模板，提供后台管理、角色管理、权限配置、数据表管理、插件功能、媒体数据保存等功能。

可应对不同场景的数据手机、信息反馈功能。

同时后端提供低代码API开发，前端开发者无需了解后端代码知识便可以实现增删查改的功能。

例如：数据收集、官网展示、个人博客、商城、新闻资讯、论坛。
